## Publish with GitHub Pages
1) Create repo rideshare-wiki
2) Upload these files to repo root
3) Settings → Pages → Deploy from branch main / root

How to push Updates
# 1. Clone (only once)
git clone https://github.com/your-username/your-repo.git
cd your-repo

# 2. Edit files locally
# (VS Code, Notepad++, etc.)

------- Update
# 3. Check changes
git status

# 4. Stage changes
git add .

# 5. Commit
git commit -m "Update web pages"

# 6. Push to GitHub
git push origin main

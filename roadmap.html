Roadmap (Phases 1–4)

A phased technology plan that grows from demo → production → analytics/ML → modern data warehouse.

Phases

Phase Focus Outcome
Phase 1 Demo Low-cost demo environment, validate UX and flows
Phase 2 Production (No SQL) + Admin Operator-ready base, admin capabilities, edge security baseline
Phase 3 Analytics + ML + Event-driven Event pipelines, ML readiness, scalable analytics
Phase 4 Modern Data Warehouse Warehouse + BI concurrency + governance baseline
Design intent: each phase adds capability without rewriting the core platform.

What stays constant

  • Driver-first governance (70–80%)
  • SaaS-first operating model
  • Hybrid pricing aligned to usage
  • Event-driven backbone

What evolves

  • Messaging: RabbitMQ → Kafka
  • Analytics: light → ML pipelines → warehouse
  • Security: baseline edge → tuned rules later
